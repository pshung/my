
# VSCode shortcut setting.

search files by name
> ctrl + p

command pallete
> ctrl + shift + p

open setting
> ctrl + ,

multi-cursor editing
> alt + click to add new cursor.

shortcut setting
> ctrl + K + ctrl + S

search references
> gr

go to definition
> gd

go to implementation
> gi

toggle zen mode
> . + z

switch files
> ctrl + tab
